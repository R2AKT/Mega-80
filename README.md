# Mega-80

Mega-80 (Mega-580) project repository.

A project to create a homebrew 8-bit microcomputer based on the K580 microprocessor kit (Intel 8080).

Проект по созданию самодельного 8-разрядного микрокомпьютера на базе микропроцессорного комплекта K580 (Intel 8080).
